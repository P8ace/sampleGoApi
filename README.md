# sampleGoApi
sample go postgres api app repository

Dependencies:
- HTTP router: [chi](https://github.com/go-chi/chi)

Dev dependencies:
- DB sql schema migrations : [goose](https://pressly.github.io/goose/)
- DB sql query compiler : [sqlc](https://docs.sqlc.dev/en/latest/index.html)
- DB postgres driver : [pgx](https://github.com/jackc/pgx/v5)
